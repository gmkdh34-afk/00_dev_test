# CLAUDE.md

이 파일은 Claude Code (claude.ai/code)가 이 저장소에서 작업할 때 참고할 수 있는 가이드를 제공합니다.

## 프로젝트 개요

매일 다른 명언을 보여주고 인터랙티브 달력 기능을 제공하는 "오늘의 명언" 단일 페이지 웹 애플리케이션입니다.

## 아키텍처

**단일 HTML 파일 구조**: 모든 애플리케이션이 `index.html`에 CSS와 JavaScript가 포함된 형태로 구성되어 있습니다 (빌드 과정이나 의존성 불필요).

**주요 컴포넌트**:
- **명언 표시 시스템**: 날짜 기반 시드를 사용하여 매일 하나의 일관된 명언을 표시
- **달력 컴포넌트**: 월 단위 네비게이션이 가능한 인터랙티브 달력
- **팝업 시스템**: 과거 날짜에 마우스를 올리면 해당 날짜의 명언을 보여주는 툴팁
- **데이터 저장**: 브라우저 localStorage를 사용하여 날짜별 명언 매핑 유지

## 실행 방법

최신 웹 브라우저에서 `index.html`을 열기만 하면 됩니다. 빌드 단계, 서버, 의존성 설치가 필요하지 않습니다.

## 기술적 세부사항

**명언 선택 로직** (index.html:297-315):
- 날짜를 시드로 사용 (`YYYY * 10000 + MM * 100 + DD`)하여 결정론적 명언 선택
- 각 날짜는 명언 배열에 대한 모듈로 연산을 통해 항상 같은 명언 반환
- 결과는 localStorage에 `YYYY-M-D` 형식의 키로 캐싱됨

**달력 렌더링** (index.html:325-376):
- 현재 월/연도 상태를 기반으로 달력 그리드를 동적으로 생성
- 과거 날짜(오늘 포함)는 `.has-quote` 클래스와 호버 이벤트 적용
- CSS Grid 레이아웃 사용 (요일별 7열)

**상태 관리**:
- `currentMonth` 및 `currentYear`: 달력 뷰 상태 추적
- `today`: 기준 날짜 (페이지 로드 시 한 번 설정)
- localStorage: 세션 간 명언 히스토리 유지

## 주요 함수

- `getQuoteForDate(date)`: 특정 날짜의 명언 반환 (캐싱 포함)
- `displayTodayQuote()`: 메인 컨테이너에 오늘의 명언 렌더링
- `renderCalendar()`: 현재 월/연도의 달력 UI 생성
- `showQuotePopup(event, date, quote)`: 호버 팝업 위치 설정 및 표시
- `hideQuotePopup()`: 팝업 툴팁 숨기기

## 명언 수정하기

명언 배열 (index.html:252-290)에는 동서양 명언 37개가 포함되어 있습니다. 추가/수정 방법:
1. `quotes` 배열을 `{ text: string, author: string }` 형태의 객체로 편집
2. 다른 변경사항 불필요 - 명언 선택 로직이 자동으로 새 배열 길이를 사용함
